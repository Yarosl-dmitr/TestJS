# Task Manager
Простой менеджер задач для демонстрации навыков разработки и тестирования.

## Функционал
- Добавление задач
- Удаление задач
- Отметка о выполнении задач

## Используемые технологии
- HTML
- CSS
- JavaScript

## Как запустить
1. Скачайте проект
2. Откройте `index.html` в браузере
